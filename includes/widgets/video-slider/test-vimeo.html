Troubleshooting

If videos don't autoplay, check:

  • Browser autoplay policies: Chrome/Safari require user interaction first
  • Console errors: Open developer tools and check for JavaScript errors
  • Network tab: Verify video files are loading successfully
  • CORS issues: Self-hosted videos need proper CORS headers
  • Video format: Ensure MP4 is properly encoded

Expected Behavior

  • MP4 video should start playing immediately (if supported)
  • Vimeo video should start playing after a brief load time
  • Both videos should loop continuously
  • Videos should be muted by default
  • Videos should scale to fill container while maintaining aspect ratio
